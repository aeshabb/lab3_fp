# Тесты

Эта папка содержит все тестовые файлы и данные для проекта.

## Структура

- **examples/** - тестовые данные в CSV формате
  - `linear_test.txt` - данные для линейной интерполяции
  - `newton_test.txt` - данные для интерполяции Ньютона
  - `complex_test.txt` - более сложные тестовые данные

- **run_tests.sh** - скрипт для запуска всех автоматических тестов
- **demo_interactive.sh** - демонстрация интерактивного режима работы
- **demo_comparison.sh** - сравнение методов интерполяции (линейная vs Ньютон)
- **test_debug.ml** - отладочные тесты (OCaml)
- **test_windows.ml** - тесты для скользящего окна (OCaml)

## Запуск тестов

### Все автоматические тесты
```bash
bash tests/run_tests.sh
```

### Демонстрация интерактивного режима
```bash
bash tests/demo_interactive.sh
```

### Сравнение методов интерполяции
```bash
bash tests/demo_comparison.sh
```

Этот скрипт покажет разницу между линейной и Ньютоном с разными параметрами.

### Отдельные тесты

#### Тест 1: Линейная интерполяция
```bash
cat tests/examples/linear_test.txt | dune exec my_lab3 -- --linear --step 0.7
```

#### Тест 2: Интерполяция Ньютона
```bash
cat tests/examples/newton_test.txt | dune exec my_lab3 -- --newton -n 4 --step 0.5
```

#### Тест 3: Сложная функция
```bash
cat tests/examples/complex_test.txt | dune exec my_lab3 -- --linear --step 1.0
```

## Создание своих тестов

Создайте файл с данными в формате CSV:
```
# mytest.txt
0 0
1 1
2 4
3 9
```

Запустите тест:
```bash
cat mytest.txt | dune exec my_lab3 -- --linear --step 0.5
```

## Форматы данных

Поддерживаются следующие разделители:
- Пробел: `0 0`
- Табуляция: `0	0`
- Точка с запятой: `0;0`
- Запятая: `0,0`

Все точки должны быть отсортированы по возрастанию x-координаты.
